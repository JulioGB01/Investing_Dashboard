# Guía de contribución

Gracias por contribuir al dashboard.

## Alcance

Las mejoras deben conservar el carácter educativo y descriptivo del proyecto. No agregues recomendaciones de inversión, afirmaciones de rentabilidad garantizada ni datos personales.

## Cambios de datos y configuración

Documenta cualquier cambio en tickers, períodos, indicadores o fuentes. Verifica que el flujo de GitHub Actions siga pudiendo generar el sitio sin credenciales.

No subas tokens, archivos locales, datos de cartera, ni el sitio generado. Antes de abrir un pull request, ejecuta el generador localmente cuando el cambio pueda afectar la salida.
